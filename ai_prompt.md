# 股票投资分析 AI Prompt 模板

```
你是一名资深买方分析师，请对 [公司名称/股票代码] 做一份基本面深度分析，涵盖以下部分：

1. 公司概况
   - 主营业务、商业模式、收入构成
   - 所处行业及产业链位置

2. 财务分析
   - 近3-5年营收、净利润、毛利率、净利率趋势
   - 现金流状况（经营性现金流是否支撑利润）
   - 资产负债表健康度（负债率、流动比率）
   - ROE/ROIC 及杜邦拆解

3. 估值分析
   - 当前 PE/PB/PS/EV-EBITDA，并与历史区间、同行业对比
   - 简要给出 DCF 或相对估值的合理区间（说明关键假设）

4. 竞争格局与护城河
   - 主要竞争对手对比
   - 护城河来源（品牌、成本、网络效应、转换成本等）

5. 增长驱动与催化剂
   - 未来1-3年主要增长点
   - 潜在催化剂（新产品、政策、并购等）

6. 风险因素
   - 行业风险、公司特有风险、宏观风险
   - 财务造假/治理风险信号（如有）

7. 结论
   - 给出一个中性、基于数据的观点（不是投资建议），标注关键不确定性
   - 明确说明哪些数据是你已知的，哪些需要用户自行核实最新数据（因为你的知识可能不是最新的）

请用简体中文回答，数据尽量给出来源或估算依据，不确定的地方明确标注"需核实"。
```

## 二、技术面 / 短线交易分析

```
请对 [股票代码] 做技术面分析，包括：

1. 当前趋势判断（日线/周线级别，均线排列、量价关系）
2. 关键支撑位与阻力位
3. 常见技术指标信号（MACD、RSI、布林带、成交量）是否出现背离或过热/超卖
4. 近期是否处于关键形态（突破、整理、头肩顶底等）
5. 短期（1-4周）情景推演：上行/下行/震荡各自的触发条件和大致空间
6. 风险提示：技术分析的局限性，需结合基本面和消息面

请明确说明你没有实时行情数据，分析基于用户提供的最新数据或截图。
```

## 三、行业对比 / 选股筛选

```
我关注 [行业名称] 板块，请帮我：

1. 列出该行业内值得关注的 5-10 家上市公司
2. 用一张对比表列出：市值、估值水平（PE/PB）、营收增速、净利率、ROE、近一年涨跌幅
3. 按"稳健型"、"成长型"、"困境反转型"分类
4. 指出行业当前所处周期阶段（导入期/成长期/成熟期/衰退期）及主要驱动因素/政策影响
5. 给出 2-3 个值得进一步研究的方向，并说明理由

数据如果超出你的知识范围或可能过时，请明确提示我需要自行核实最新数字。
```

## 四、持仓组合风险评估

```
以下是我的持仓（股票代码/名称、仓位占比、成本价）：
[粘贴持仓明细]

请帮我分析：

1. 行业/风格集中度（是否过度集中在某几个行业或风格因子）
2. 相关性风险（持仓之间是否高度联动，缺乏分散效果）
3. 单一个股风险敞口是否过高
4. 结合当前宏观环境（利率、汇率、行业政策等，请先说明你的信息截止时间），指出组合可能面临的主要风险场景
5. 给出几个可以考虑的再平衡方向（不构成具体投资建议，仅作分散化思路参考）

请以中立客观的方式呈现，避免给出确定性的买卖建议，多用"可以考虑""值得关注"等表述。
```

## 五、快速盘前/盘后简报

```
请基于以下信息（[粘贴新闻/公告/财报原文]），用不超过 300 字总结：

1. 核心事件是什么
2. 对公司基本面的实质影响（利好/利空/中性，及理由）
3. 市场可能的解读角度（可能有分歧的地方）
4. 需要持续跟踪的后续信息点

语言简洁，不做绝对化判断。
```
